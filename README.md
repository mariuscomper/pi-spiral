# π Spiral — 768 Digits to the Feynman Point

An interactive visualization of the first 768 decimal digits of π, mapped onto an Archimedean spiral. Each digit is colour-coded (0–9), and the **Feynman Point** — six consecutive 9s beginning at the 762nd decimal place — pulses in gold.

**[View live →](https://mariuscomper.github.io/pi-spiral/)** *(update with your GitHub Pages URL)*

## Features

- **Archimedean spiral layout** — digits flow outward from π at the centre
- **Interactive** — hover (desktop) or tap (mobile) any dot to see its position and value
- **Feynman Point highlighted** — gold rings animate around positions 762–767
- **Fully responsive** — adapts to any screen size, retina-ready (up to 3× DPR)
- **Touch-optimised** — larger hit targets and touch events for mobile
- **Zero dependencies** — single self-contained HTML file, pure Canvas API
- **Dark theme** — designed for dark backgrounds

## Usage

Open `index.html` in any modern browser. That's it.

To host on GitHub Pages:
1. Rename `pi-spiral.html` → `index.html`
2. Push to a repo
3. Enable Pages in repo settings (deploy from main branch)

## The Feynman Point

Richard Feynman once remarked that he'd like to memorise π up to the point where six 9s appear in sequence (position 762), so he could recite it and end with "nine nine nine nine nine nine and so on" — implying π might be rational. It isn't, of course. But the joke lands.
